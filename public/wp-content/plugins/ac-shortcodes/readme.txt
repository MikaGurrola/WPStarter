Convert any Divi Modules into a shortcode and use them in any module editors


== Description ==

AC Shortcodes is a very simple plugin. It will allow you to convert any Divi module into a shortcode which you can use into any other module editor and render the output as you design in the shortcode maker.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Move to Divi Admin menu and find the `AC Shortcodes Admin Sub Menu`. If you're using Extra theme, the sub-menu will be appended to teh Extra theme options. For other themes, the AC Shortcodes plugin sub-menu will be added in the Appearance admin menu
